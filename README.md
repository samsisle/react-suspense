###### Demo: [https://emojiest.now.sh](https://emojiest.now.sh)

**emojiest** is a simple source for all your favorite emojis. Copy your favorite emojis with ease.

## Installation

```bash
$ git clone https://github.com/samsisle/emojiest.git
$ cd emojiest
$ yarn && yarn start
```
